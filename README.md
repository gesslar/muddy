## Immediate Invocation

```shell
# pnpm
pnpx @gesslar/muddy --help

# npm
npx @gesslar/muddy --help
```

## Install as a dependency, if you want, you don't have to

```shell
# pnpm
pnpm add -d @gesslar/muddy

#npm
npm i -d @gesslar/muddy
```
